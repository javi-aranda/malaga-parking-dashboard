Este dashboard es parte de un proyecto para la visualización de datos de ocupación de parkings en Málaga.

Los datos son obtenidos del repositorio https://github.com/javi-aranda/malaga-parking-data, que a su vez los recopila del endpoint de consulta de datos abiertos del Ayuntamiento de Málaga desde mayo de 2022.

En enero de 2024 el formato de los datos cambió, previamente había sensores de parkings que no estaban funcionando correctamente y se han descartado para su análisis.

El código fuente de esta herramienta, así como la base de datos, está disponible en https://github.com/javi-aranda/malaga-parking-dashboard.
